# 🚀 بناء APK عبر GitHub Actions — دليل سريع

## الفكرة ببساطة

بدل ما تبني الـ APK على جهازك (يحتاج Android Studio و SDK ضخم)، GitHub Actions **يبني لك الـ APK على سيرفرات سحابية مجانية**، والنتيجة: ملف `app-release.apk` تنزّله على موبايلك وخلاص.

---

## 📋 الخطوات

### 1) إنشاء حساب GitHub (لو ما عندك)
- روح **github.com** وسجّل (مجاني)
- **مهم:** إيميلك لازم يكون متاح لأنه هيتبعت عليه إشعارات البناء

### 2) إنشاء مستودع جديد
1. من حسابك في GitHub اضغط **+** (أعلى يمين) → **New repository**
2. اسم المستودع: `alamin-pharma` (أو أي اسم)
3. اختر **Public** (مهم عشان Actions المجاني يشتغل)
4. **لا تضف** README أو .gitignore (عندنا بالفعل)
5. اضغط **Create repository**

### 3) رفع المشروع

#### الطريقة A — عبر الموقع (سهلة للمبتدئين)
1. في صفحة المستودع الجديد، اضغط **uploading an existing file** (الرابط الموجود تحت "Quick setup")
2. اسحب **كل محتويات** مجلد `alamin-pharma/` (اللي فكيته من ZIP) إلى منطقة الرفع
3. **مهم:** اسحب المحتويات **الداخلية** للمجلد، مش المجلد نفسه
4. اكتب commit message مثل: `first commit`
5. اضغط **Commit changes**

#### الطريقة B — عبر Git (لو تعرف)
```bash
cd alamin-pharma
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/اسم-المستخدم/alamin-pharma.git
git push -u origin main
```

### 4) تشغيل البناء
1. في مستودعك على GitHub، اضغط تبويب **Actions** (أعلى الصفحة)
2. هتلاقي workflow اسمه **"Build Android APK"** في القائمة الجانبية
3. اضغط عليه → اضغط **Run workflow** → **Run workflow** (الزر الأخضر)
4. انتظر **5-10 دقائق** (أول مرة بيحمل Gradle dependencies)

### 5) تنزيل الـ APK
1. بعد ما يخلّص، اضغط على اسم الـ run اللي ظهر
2. انزل لتحت لتبويب **Artifacts**
3. هتلاقي ملف اسمه:
   - **`alamin-pharma-release`** ← هذا للـ Release (موقّع، يشتغل على أي جهاز) ✅
   - **`alamin-pharma-debug`** ← هذا للـ Debug (نسخة اختبار)
4. اضغط عليه → بيبدأ تنزيل ملف ZIP
5. **افتح الـ ZIP** واستخرج `app-release.apk`

### 6) تثبيت الـ APK على موبايلك
1. ابعث `app-release.apk` لنفسك عبر واتساب/تليجرام/إيميل، أو انقله بكابل USB
2. افتح الملف على الموبايل
3. وافق على "تثبيت من مصادر غير معروفة" (مرة واحدة)
4. **تثبيت** ← خلاص ✅ 🎉

---

## ⚠️ لو البناء فشل

- اضغط على اسم الـ run الأحمر
- اضغط على الـ step اللي فشل (الأحمر)
- انسخ نص الخطأ وابعثه لي — بحلّه لك

---

## 💡 بديل أسرع (Appetize-style)

لو عندك رابط عام للملفات، ممكن نرفع الـ APK الجاهز (لو في) ويطلع رابط مباشر. بس GitHub Actions هو الأضمن والأسرع.
